## <b>두 수의 곱</b>
### ✏️문제설정
정수 `num1, num2`가 매개변수 주어집니다. `num1과 num2를 곱한 값을 return` 하도록 solution 함수를 완성해주세요.
### ✏️답(JAVA)
```java
class Solution {
    public int solution(int num1, int num2) {
        return num1*num2;
    }
}
```
### ✏️답(Python)
```python
def solution(num1, num2):
  return num1*num2
```
