## <b>두 수의 차</b>
### ✏️문제설정
정수 `num1`과 `num2`가 주어질 때, `num1에서 num2를 뺀 값을 return`하도록 soltuion 함수를 완성해주세요.
### ✏️답(JAVA)
```java
class Solution {
    public int solution(int num1, int num2) {
        return num1-num2;
    }
}
```
### ✏️답(Python)
```python
def solution(num1, num2):
  return num1-num2
```
