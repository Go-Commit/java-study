## <b>두 수의 합</b>
### ✏️문제설정
정수 `num1`과 `num2`가 주어질 때, `num1과 num2의 합`을 return하도록 soltuion 함수를 완성해주세요.
### ✏️답(JAVA)
```java
class Solution {
    public int solution(int num1, int num2) {
        return num1+num2;
    }
}
```
### ✏️답(Python)
```python
def solution(num1, num2):
  return num1+num2;
```
