### 중앙값 구하기

```JAVA
import java.util.Arrays;

class Solution {
    public int solution(int[] array) {
        Arrays.sort(array);
        return array[array.length/2];
    }
}
```

### 짝수는 싫어요

```JAVA
class Solution {
    public int[] solution(int n) {
        
        int count = 0;
        int idx = 0;
        
        for(int i=1; i<=n; i++)
        {
            if(i%2!=0) count++;
        }
        
        int[] answer = new int[count];
        
        for(int i=0; i<=n; i++)
        {
            if(i%2!=0)
            {
                answer[idx]=i;
                idx++;
            }
        }
        
        return answer;
    }
}
```

class Solution {
    public int solution(int n) {
        int answer = 0;
        for(int i=1; i<=n; i++){
            if(i * 6 % n == 0){
                answer = i;
                break;
            }
        }
        return answer;
    }
}

### 피자 나눠 먹기 (3)

```JAVA
class Solution {
    public int solution(int slice, int n) {
        int diviNum = n / slice;
        int restNum = n % slice;

        if (restNum > 0) {
            diviNum += 1;
        }
        
        int answer = diviNum;
        
        return answer;
    }
}
```

### 배열의 평균값

```JAVA
class Solution {
    public double solution(int[] arr) {
        double answer = 0;
        double sum = 0;
        for (int i=0;i<arr.length;i++){
            sum += arr[i];
        }
        answer = sum / arr.length;                    
        return answer;            
    }
}
```
