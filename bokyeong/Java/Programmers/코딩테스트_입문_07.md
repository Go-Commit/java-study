### 삼각형의 완성조건 (2)

```JAVA
class Solution {
    public int solution(int[] sides) {
        int answer = 0;

        if (sides[0] > sides[1]) {
            for (int i=sides[0]-sides[1]+1; i<sides[0]+sides[1]; i++) {
                answer++;
            }
        } else {
            for (int i=sides[1]-sides[0]+1; i<sides[1]+sides[0]; i++) {
                answer++;
            }
        }

        return answer;
    }
}
```

### 겹치는 선분의 길이

```JAVA
import java.util.HashMap;
import java.util.Map;

class Solution {
    public int solution(int[][] lines) {
        int answer = 0;

        // Map<key, value>
        Map<String, Integer> map = new HashMap<>();

        for (int[] line : lines) {
            int tempMin = Math.min(line[0], line[1]);
            int tempMax = Math.max(line[0], line[1]);
            for (int i=tempMin+1; i<tempMax+1; i++) {
                String str = (i - 1) + "/" + i;
                // getOrDefault(Object key, V DefaultValue)
                // 찾는 키가 존재한다면 찾는 키의 값을 반환하고 없다면 기본 값을 반환
                map.put(str, map.getOrDefault(str, 0) + 1);
            }
        }

        for (Integer value : map.values()) {
            if (value > 1) answer++;
        }

        return answer;
    }
}
```
