# 상속


***클래스상속***
```
class 자식클래스 extends 부모클래스 {
  //필드
  //생성자
  //메소드

}
```

#### 7.2 클래스 상속 예제

```

public class CellPhone {
	//필드
	String model;
	String color;
	
	//생성자
	
	//메소드
	
	void powerOn() { System.out.println("전원을 켭니다."); }
	void powerOff() { System.out.println("전원을 끕니다."); }
	void bell() { System.out.println("벨 울림."); }
}






```
